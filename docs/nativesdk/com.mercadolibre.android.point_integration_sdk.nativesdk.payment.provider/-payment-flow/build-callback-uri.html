This method build a valid URI to open specific activity using deeplink strategy. To configure this call to an activity, you must configure the deeplink in the AndroidManifest where the activity that will answer the call is registered.

Return

the URI formed with the information entered.

Parameters

callback

Uri value to call deeplink e.g. yourHost://myapp/test.

methodCallback

Parameter to identify if this URI is to success, error or another custom response.
