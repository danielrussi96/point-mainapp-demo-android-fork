launchPaymentFlowActivity

abstract fun launchPaymentFlowActivity(amount: String, description: String? = StringUtils.EMPTY, intentSuccess: Uri, intentError: Uri, context: Context, paymentMethod: String? = null)
Content copied to clipboard

This method launch the payment flow using the SmartPOS app.

Parameters

amount

String value to set payment amount, Only supports numbers (0...9) and decimal separator (.). Supports only two decimal numbers. Minimum amount: 1.0 Default, 0.01 in Brazil and 5.0 in Mexico. Maximum amount: 50000.0 Default, 1000000.0 in Argentina and 70000.0 in Brazil.

description

Optional field to send payment description, This field supports maximum 25 characters A-Z,a-z, 0-9.

intentSuccess

Uri to call success screen, this URI is used to form a deeplink to send a success activity. To make this URI you can use launchPaymentFlowIntent() method.

intentError

Uri to call error screen, this URI is used to form a deeplink to send a error activity. To make this URI you can use launchPaymentFlowIntent() method.

context

app context necessary to launch the activity.
